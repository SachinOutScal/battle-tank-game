Battle Tank Game 


The game uses art assets from the unity tank tutorial game. 

The code has been rewritten from scatch using the MVC design pattern.  
A singleton Service component has been added on top of MVC as managerial class.
Other Design patterns used are state machine, observor, object pooling
Async await another interesting feature from new C# library is also used
